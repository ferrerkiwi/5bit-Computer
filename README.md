# 5bit-Computer
This project is a complete simulation of a functional 5-bit CPU architecture built within NI Multisim. It demonstrates the fundamental principles of computer organization, transitioning from raw logic gates to a system capable of executing machine-level instructions.

By designing this system, I implemented the core components of the von Neumann architecture, including a custom Instruction Set Architecture (ISA), an Arithmetic Logic Unit (ALU), and synchronized control logic.
